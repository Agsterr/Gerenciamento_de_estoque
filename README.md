# 🚀 Sistema de Gerenciamento de Estoque  

Este projeto é um **Sistema de Gerenciamento de Estoque** desenvolvido para facilitar o controle de produtos, oferecendo funcionalidades como cadastro, edição, exclusão e geração de relatórios.  

## 📌 Funcionalidades  
✔️ Cadastro, edição e exclusão de produtos  
✔️ Controle de entrada e saída de estoque  
✔️ Pesquisa e filtragem de produtos  
✔️ Autenticação e controle de acesso  
✔️ Geração de relatórios em PDF  

## 🛠️ Tecnologias Utilizadas  

### **Back-end**  
- Java 17 + Spring Boot  
- Spring Security + JWT  
- JPA/Hibernate  
- PostgreSQL  

### **Front-end**  
- Angular  
- TypeScript  
- HTML, CSS  

## 🚀 Como executar o projeto  

### **Pré-requisitos**  
- Node.js + Angular CLI  
- Java 17+  
- PostgreSQL  
- Maven  

### **Rodando o Back-end**  
1. Clone o repositório:  
   ```bash
   git clone https://github.com/Agsterr/Gerenciamento_de_estoque.git
   cd Gerenciamento_de_estoque
   ```
2. Configure o banco de dados no `application.properties`  
3. Execute o projeto com:  
   ```bash
   mvn spring-boot:run
   ```

### **Rodando o Front-end**  
1. Acesse a pasta do front-end  
   ```bash
   cd Gerenciamento_de_estoque_front
   ```
2. Instale as dependências  
   ```bash
   npm install
   ```
3. Inicie o projeto  
   ```bash
   ng serve
   ```
